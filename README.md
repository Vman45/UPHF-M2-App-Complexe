# UPHF-M2-App-Complexe
Domotic application to watch and act in case of a problem in the house of an old person.

## Folders 
**Code Raspberry Pi** : contains the code for de Domotic captors Zwave and the code to listen the UDP Sensor application on the smartphone.\n
**IA** : contains the test code to implement an AI on a server.\n
**Skill Alexa** : contains all the code for the Alexa Skill which send a mail in case of problem.\n\n

You'll find the presentation PowerPoint of the project named "Pr√©sentation_App_Complexe" in that git.\n
